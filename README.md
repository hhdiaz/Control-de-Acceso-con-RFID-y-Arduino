# Control-de-Acceso-con-RFID-y-Arduino
Este proyecto utiliza un Arduino MEGA y un módulo RFID para crear un sistema de control de acceso  para puertas. El dispositivo lee tarjetas RFID, verifica la autorización y controla un servo motor para  abrir o cerrar la cerradura de la puerta. 

Objetivo:
Desarrollar un sistema de control de acceso económico y personalizable que utilice tecnología RFID 
para gestionar la entrada a un espacio determinado.

Componentes 
• Arduino MEGA 
• Módulo RFID RC522 
• Pantalla LCD con módulo I2C 
• Servo motor 
• Mecanismo de la cerradura
